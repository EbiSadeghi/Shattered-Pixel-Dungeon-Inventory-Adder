# Setup
- Install Shattered Pixel Dungeon
  - This script was test using Snap on Linux
- Start a game, die at least once
- Clone main.py from this repo

# Usage
- Start a new game
- Exit the game and close Shattered Pixel Dungeon
- Navigate to your game data location:
  - **Example:** /home/ubuntu/snap/shattered-pixel-dungeon/639/.local/share/.shatteredpixel/shattered-pixel-dungeon
    - ***Note:*** *639 is the game version number in the example above*
    - For other operating systems the wiki is a decent guide: https://pixeldungeon.fandom.com/wiki/Save_files
- Open the game folder you wish to modify
-   **Example:** game2
-   Extract the game.dat file, you will have a "game" file generated with no file extension
-   Run the script in the same directory as the "game" file
-   Move the script-generated game.dat into the game# folder you chose earlier
